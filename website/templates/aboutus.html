SaleSync

Version: 1.0 | Developed in Visual Studio Code using Python, Flask, and SQLite 3

SYSTEM DESCRIPTION

The system for The Crazy Fish Man (TCFM) integrates inventory management, a user-friendly
point of sale (POS) interface for generating receipts and handling transactions, and allows users
to generate inventory and transaction reports. It features three access levels: Pending (access to
Help and About pages), Staff (adds Cashier Operations and Inventory pages), and Admin
(adds user management and transactions pages). Security is ensured with two-factor
authentication via email and hashed passwords.

Products in the inventory and transactions are sorted using the Quick Sort Algorithm.
